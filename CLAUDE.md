# Banda Lady Hooke — Repo de Conteúdo

Repositório de conteúdo do site institucional da **Banda Lady Hooke**.
Gerado a partir do template [code7tecbr/my-sites](https://github.com/code7tecbr/my-sites).

## Como funciona

Qualquer push na branch `main` dispara `.github/workflows/deploy.yml`,
que chama `code7tecbr/my-sites-action@v1` e publica o site no Cloudflare.

## O que editar

Todo o conteúdo do site fica em `content/`. Nunca edite código Vue/Nuxt — ele fica no template privado.

Use `/content-guide` para obter orientações detalhadas sobre cada arquivo de conteúdo.

## Idioma

- Documentação: português do Brasil
- Chaves dos JSONs: inglês (não alterar)
